# Mon-Repas Client Mobile

Application mobile **Android / iOS** (Flutter) pour les clients de Mon-Repas.
Reprend l'intégralité des fonctionnalités de la webapp client (`../mon-repas_client`) et consomme la même API (`../mon-repas_api`, port 3502).

📄 **Source de vérité : [SPECIFICATIONS.md](./SPECIFICATIONS.md)** — stack, architecture, thème, écrans, contrat API, règles métier et roadmap.

## Démarrage rapide (à venir)

Le scaffold Flutter sera créé en Phase 0 de la roadmap (voir SPECIFICATIONS.md §13).

```bash
flutter run --dart-define=MONREPAS_API_URL=http://localhost:3502
```
